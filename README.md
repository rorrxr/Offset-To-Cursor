# Offset-To-Cursor
페이징 기법 방식이 오프셋 기반에서 커서 기반으로 변환하는 과정을 담은 예시 프로젝트입니다.

# 프로젝트 기술 스택

- Spring Boot 3.x
- Java 21
- Spring Data JPA
- MySQL
- H2 Database
- QueryDSL
